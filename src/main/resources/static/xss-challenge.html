⚡ XSS Stored

Défi #4 - Cross-Site Scripting
Exploitez une vulnérabilité XSS Stored pour voler les cookies d'un administrateur.

🎯 Objectif du Défi :

Injectez du code JavaScript malveillant dans un commentaire pour voler le cookie administrateur contenant le flag.

💡 Instructions

1. Postez un commentaire malveillant avec du code JavaScript

2. Simulez la visite d'un administrateur (bouton ci-dessous)

3. Rechargez les commentaires

4. Votre payload XSS s'exécutera

5. Exfiltrez le cookie vers votre serveur HTTP externe
